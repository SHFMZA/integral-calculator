# Программа для вычисления площади через определенный интеграл

## Описание

Программа для вычисления площади фигуры, ограниченной графиком функции y = f(x), осью OX и вертикальными прямыми x = a и x = b.

## Возможности

- Вычисление площади с помощью различных методов численного интегрирования
- Графический интерфейс на tkinter
- Визуализация графика функции и заливка области
- Обработка ошибок ввода

## Методы интегрирования

1. **Метод прямоугольников** - средние прямоугольники
2. **Метод трапеций** - трапеции между точками
3. **Метод Симпсона** - параболические аппроксимации

## Требования

- Python 3.6+
- tkinter (входит в стандартную библиотеку Python)
- matplotlib
- numpy

## Установка зависимостей

```bash
pip install matplotlib numpy
```

## Запуск программы

```bash
python integral_calculator.py
```

## Интерфейс программы

### Основные элементы:
- **Функция f(x)** - поле для ввода функции
- **Нижний предел (a)** - левая граница интегрирования
- **Верхний предел (b)** - правая граница интегрирования
- **Количество разбиений (n)** - точность вычислений
- **Метод интегрирования** - выбор алгоритма
- **Кнопка "Вычислить"** - запуск расчета
- **Результат** - вывод площади
- **График** - визуализация функции

## Примеры использования

### Пример 1: Простая функция

- Функция: `x**2`
- a = 0, b = 2
- Метод: Трапеций
- n = 1000
- Результат: ~2.6667

### Пример 2: Тригонометрическая функция

- Функция: `sin(x)`
- a = 0, b = π
- Метод: Симпсона
- n = 101 (нечетное)
- Результат: ~2.0

### Пример 3: Сложная функция

- Функция: `exp(x)`
- a = 0, b = 1
- Метод: Прямоугольников
- n = 500
- Результат: ~1.7183

## Поддерживаемые функции

Программа поддерживает:
- Арифметические операции: `+`, `-`, `*`, `/`, `**`
- Тригонометрические: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`
- Экспоненциальные: `exp`, `log`, `log10`
- Другие: `sqrt`, `abs`, `min`, `max`
- Константы: `pi`, `e`

## Ошибки и ограничения

- Функция должна быть определена на всей области [a, b]
- Для метода Симпсона n должно быть нечетным
- a должно быть меньше b
- n должно быть положительным
